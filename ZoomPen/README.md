# Puti-AI Z-Pen 螢幕畫筆與放大工具

這是一款對齊 Windows 知名工具 ZoomIt 的輕量化螢幕放大與繪圖標註工具，支援 **Windows** 與 **macOS** 雙系統。

---

## 📦 安裝與下載指引

### Windows 系統
* **運行方式**：直接下載並執行單一可攜式檔案 `Puti-AI-Z-Pen.exe`，無須安裝。

### macOS 系統
請根據您的 Mac 處理器晶片下載對應的安裝檔：
* **Intel 處理器舊型 Mac**：下載檔名含 `-x64` 的 dmg 檔 (例如：`Puti-AI-Z-Pen-1.0.0-x64.dmg`)。
* **Apple Silicon (M1/M2/M3) 晶片 Mac**：下載檔名含 `-arm64` 的 dmg 檔 (例如：`Puti-AI-Z-Pen-1.0.0-arm64.dmg`)。

#### 🍏 macOS 首次啟用設定步驟：
1. **解鎖開發者限制**：
   * 雙擊打開 `.dmg`，將 App 拖曳至「應用程式」資料夾。
   * 首次點擊開啟若提示「無法驗證開發者」，請前往 Mac 的 **「系統設定」->「隱私權與安全性」**，滑到下方點擊 **「強制打開」**。
2. **開啟螢幕錄製權限**（*非常重要，否則放大畫面會呈現全黑*）：
   * 開啟 App 後，按下 `Cmd + 1` 或點擊工具列的放大鏡。
   * 系統提示需要「螢幕錄製」權限時，點選「打開系統設定」，在列表中啟用 **Puti-AI Z-Pen**。
   * **啟用權限後，請將 App 關閉重啟**，即可正常使用。

## ⌨️ 雙平台快捷鍵對照表

### 核心模式切換

| 功能說明 | Windows 快捷鍵 | macOS 快捷鍵 |
| :--- | :--- | :--- |
| **進入 Zoom (放大鏡模式)** | `Ctrl + 1` | `Cmd + 1` |
| **進入 Draw (靜態畫筆模式)** | `Ctrl + 2` | `Cmd + 2` |
| **進入 Live Zoom (即時操作放大)** | `Ctrl + 4` | `Cmd + 4` |
| **退出當前模式 / 關閉畫筆** | `Escape` | `Escape` |

*註：在 Zoom 模式下點擊滑鼠左鍵，會直接鎖定當前放大畫面進入繪圖狀態。*

### 🎨 畫筆標註快捷鍵 (適用於 Draw 模式 與 Zoom 繪圖狀態)

#### 1. 顏色快速切換 (直接按鍵盤字母)
* **`R`**：紅色 (`#ff4444`)
* **`G`**：綠色 (`#44ff44`)
* **`B`**：藍色 (`#4488ff`)
* **`Y`**：黃色 (`#ffdd00`)
* **`O`**：橘色 (`#ff8800`)
* **`P`**：粉紅色 (`#ff44bb`)

#### 2. 幾何圖形繪製 (按住修飾鍵並拖曳滑鼠)
* **直接拖曳**：自由曲線 (Pen)
* **`Shift` + 拖曳**：**直線**
* **`Ctrl` + 拖曳**：**長方形**
* **`Tab` + 拖曳**：**橢圓形**
* **`Ctrl` + `Shift` + 拖曳**：**箭頭**

#### 3. 螢幕黑白板背景
* **`W`**：切換為 **純白板** / 恢復原螢幕
* **`K`**：切換為 **純黑板** / 恢復原螢幕

#### 4. 文字輸入
* **`T`**：切換至**文字輸入模式（文字靠右對齊）**，在畫布點擊即可輸入文字。
* **`Shift + T`**：切換至**文字輸入模式（文字靠左對齊）**。
* 輸入框失去焦點 (`blur`) 或按 `Enter` 提交文字；按 `Esc` 取消輸入。

#### 5. 筆刷粗細與橡皮擦
* **`E`**：切換為 **單線橡皮擦模式**，直接點擊畫布上的任一線條即可擦除該物件。
* **`Ctrl + Z`** (Mac 為 **`Cmd + Z`**)：復原上一步。
* **`Ctrl + E`** (Mac 為 **`Cmd + E`**)：清除畫布上所有繪圖。
* **`Ctrl` + 滾動滾輪** 或 **`Ctrl` + 上下方向鍵** (Mac 為 **`Cmd`**)：微調筆刷粗細 (1px - 50px)。

## 🚀 開發者工作流

如果你想在本地開發或更新專案：

### 1. 複製專案與開發
```bash
git clone https://github.com/puti001/teach_tools.git
cd teach_tools/ZoomPen
npm install
npm start
```

### 2. 打包二進位檔案
* **Windows Portable EXE**：`npm run build:portable`
* **macOS 雙架構 DMG**：在 push 到 GitHub 後，專案已串接 GitHub Actions 工作流。系統會自動在 macOS 雲端環境完成編譯，請直接至 Actions 標籤頁面下載最新的 Artifact `ZoomPen-macOS` 即可。

### 3. 一鍵同步至 GitHub
直接在 PowerShell 執行部署腳本：
```powershell
./deploy_zoompen.ps1
```
*(腳本會自動排除 node_modules 等多餘檔案，並自動將 macOS 編譯工作流移動到 GitHub 倉庫的根目錄下以供 Actions 偵測運作。)*
