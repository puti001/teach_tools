# Puti-AI Z-Pen 螢幕畫筆與放大工具

這是一款對齊 ZoomIt 的輕量化螢幕放大與繪圖標註工具。

---

## 🚀 快速開始 (後續開發工作流)

如果您換了電腦或重新拉取專案，請遵循以下步驟進行開發：

### 1. 複製專案
```bash
git clone https://github.com/puti001/teach_tools.git
cd teach_tools/ZoomPen
```

### 2. 安裝套件
（因為 `node_modules` 已被排除上傳，首次執行前需重新安裝）
```bash
npm install
```

### 3. 本機測試
```bash
npm start
```

### 4. 重新打包成單一可攜式 EXE 檔案
```bash
npm run build:portable
```
打包後的檔案會存放在 `dist/Puti-AI-Z-Pen.exe`。

### 5. 一鍵同步備份到 GitHub
在 PowerShell 中直接執行部署腳本（或請 AI 助手為您執行）：
```powershell
./deploy_zoompen.ps1
```

## ⌨️ 快捷鍵與功能說明

### 雙大模式
* **Ctrl + 1**：進入 **Zoom（縮放）模式**，可滾動滾輪放大縮小。
  * **點擊左鍵**：直接在放大畫面上進入**繪圖狀態**，共享下方所有繪圖快捷鍵。
* **Ctrl + 2**：進入 **Draw（靜態繪圖）模式**，直接在當前螢幕截圖上標註。
* **Ctrl + 4**：進入 **Live Zoom（即時操作縮放）模式**，滑鼠與放大畫面游標完美重合，可一邊放大一邊正常操作電腦。
* **Escape**：退出任何模式。

---

### 🎨 核心繪圖標註快捷鍵 (適用於 Draw 模式 與 Zoom 繪圖狀態)

#### 1. 顏色快速切換 (直接按鍵盤字母)
* **R**：紅色 (`#ff4444`)
* **G**：綠色 (`#44ff44`)
* **B**：藍色 (`#4488ff`)
* **Y**：黃色 (`#ffdd00`)
* **O**：橘色 (`#ff8800`)
* **P**：粉紅色 (`#ff44bb`)

#### 2. 幾何圖形輔助 (按住修飾鍵拖曳滑鼠)
* **直接拖曳**：自由曲線 (Pen)
* **Shift + 拖曳**：**直線**
* **Ctrl + 拖曳**：**長方形**
* **Tab + 拖曳**：**橢圓形**
* **Ctrl + Shift + 拖曳**：**箭頭**

#### 3. 螢幕黑白板背景
* **W**：切換為 **白板** (純白背景) / 恢復原螢幕
* **K**：切換為 **黑板** (純黑背景) / 恢復原螢幕

#### 4. 文字輸入
* **T**：切換至**文字模式（靠右對齊）**，在畫布點擊即可輸入文字。
* **Shift + T**：切換至**文字模式（靠左對齊）**。
* 輸入框失去焦點 (`blur`) 或按 `Enter` (無 Shift) 提交文字；按 `Esc` 取消。

#### 5. 清除與修改
* **E**：切換為 **橡皮擦模式**，直接點擊畫布上的任一線條即可「單線擦除」。
* **Ctrl + Z**：復原上一步。
* **Ctrl + E**：清除畫布上所有繪圖。
* **Ctrl + 滑鼠滾輪** 或是 **Ctrl + 上下方向鍵**：微調筆刷粗細 (1px - 50px)。
