Puti-AI 簡易文本簡報產生器

貼上文本，以空行分段，每段就是一頁投影片（第一行為標題）

使用說明

基本操作

  • 將文本貼入編輯區，使用空行分隔不同頁面
  • 每個段落的第一行會自動成為該頁標題
  • 點擊「播放簡報」開始播放

播放控制

  • 使用畫面上方的按鈕或下拉選單切換頁面
  • 鍵盤快捷鍵：
    • → / 空格鍵 / Enter：下一頁
    • ← / Backspace：上一頁
    • ESC：結束播放
  • 觸控裝置可左右滑動切換頁面

簡報顯示

  • 簡報畫面可自由調整大小：
    • 拖曳簡報框右下角可調整尺寸
    • 文字會自動隨簡報大小縮放
    • 保持 4:3 比例以確保最佳顯示效果
  • 文字排版會依照原始文本的分行方式呈現

資料管理

  • 編輯器內容會自動儲存在本機
  • 匯出功能：將目前的簡報內容儲存為檔案
  • 匯入功能：讀取先前儲存的簡報檔案
