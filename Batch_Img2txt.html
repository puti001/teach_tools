Puti-AI 智慧批次拉圖進文教學

百頁成果報告書排版救星！Google 文件全自動精準置圖法

為什麼需要這個方法？

製作百頁報告時，若手動將一百多張圖片拖入文件中，會遇到以下噩夢：

  • 圖片太大撐破版面，導致軟體嚴重卡頓。
  • 排版全亂，圖片不知道飛到哪一頁。
  • 手動調整尺寸與對齊，耗費數小時。

這套「標籤預留位法 + GAS 自動化」，能讓圖片精準降落到您指定的位置，並且自動縮放、整齊劃一！

步驟一：圖片預處理（統一寬度與命名）

保命符！千萬不要把原始大圖直接丟進去。

  1. 統一縮小： 強烈推薦使用專屬線上工具 Puti-AI 圖片瘦身/轉PDF機 ，免安裝直接線上批次處理。
    • 建議：寬度統一設定為 800px（半頁圖文並茂最舒適），品質設定 80%。
    • 目的：確保排版符合 Z 字型視覺穩定感，並大幅降低文件體積防止卡頓。
    • 備案：您也可以使用單機批次縮圖軟體（如 Caesium 或 FastStone）。
  2. 規律命名： 請用流水號命名圖片，例如：001.jpg、002.png、003.jpg。
  3. 上傳雲端： 將處理好的圖片，放進 Google 雲端硬碟的一個「專屬資料夾」。
💡 取得資料夾 ID： 開啟該資料夾，網址列 `folders/` 後面那串亂碼就是 ID，先複製備用。

步驟二：在文件中植入「標籤預留位」

在撰寫報告時，只要遇到需要放圖的地方，直接輸入獨一無二的標籤。

規則： 標籤格式必須為 {{IMG_檔名}}。例如檔名是 001.jpg，標籤就是 {{IMG_001}}。

這樣做能保證文字在哪，圖片就精準卡在哪，絕不跑位。

步驟三：執行 GAS 自動化腳本

文字與標籤都完成後，讓自動化腳本幫您一鍵置圖！

  1. 在 Google 文件上方選單點選 「延伸功能」 > 「Apps Script」。
  2. 刪除原本的程式碼，將下方代碼複製貼上。
  3. 將程式碼第二行的 您的雲端硬碟資料夾ID 替換為步驟一取得的 ID。
  4. 按下上方的 「執行 (Run)」，並同意授權。
function replaceTagsWithImages() {
  // 1. 請將下方單引號內的文字替換為您的資料夾 ID
  var folderId = '您的雲端硬碟資料夾ID'; 
  
  var doc = DocumentApp.getActiveDocument();
  var body = doc.getBody();
  var folder = DriveApp.getFolderById(folderId);
  var files = folder.getFiles();
  
  // 建立圖片索引
  var imageMap = {};
  while (files.hasNext()) {
    var file = files.next();
    var fileName = file.getName().split('.')[0]; 
    imageMap[fileName] = file;
  }

  // 搜尋標籤並替換
  for (var imgKey in imageMap) {
    var tag = '{{IMG_' + imgKey + '}}';
    var found = body.findText(tag);
    
    while (found) {
      var element = found.getElement();
      var text = element.asText();
      var start = found.getStartOffset();
      var end = found.getEndOffsetInclusive();
      
      var imageBlob = imageMap[imgKey].getBlob();
      var parent = element.getParent();
      var index = parent.getChildIndex(element);
      
      // 插入圖片
      var newImage = parent.insertImage(index + 1, imageBlob);
      
      // 自動縮放 (此處設定顯示寬度為 450px)
      var width = newImage.getWidth();
      var height = newImage.getHeight();
      var newWidth = 450; 
      newImage.setWidth(newWidth);
      newImage.setHeight(height * (newWidth / width));
      
      // 刪除標籤
      text.deleteText(start, end);
      
      found = body.findText(tag, found);
    }
  }
}

完成與最後微調

回到 Google 文件，您會發現所有標籤都已經變成排列整齊的圖片了！最後您可以做快速檢查：

  • 若圖片卡在頁尾，按 Ctrl + Enter 強制換頁。
  • 若要設定圖說，請善用「樣式」功能統一格式，方便後續自動產生圖表目錄。
