Puti-AI 簡報自動化產線教學

150 頁簡報噩夢終結者！結合 Gemini 結構化產出與 Google Apps Script (GAS)，半小時內完成原本需要三天的「大綱轉投影片 + 自動置圖」體力活。

1

請 Gemini 產出「結構化大綱」

不要只讓 AI 產出純文字。利用精確的提示詞，要求 Gemini 將簡報內容格式化為表格，這是自動化的基石。

💡 提示詞範例 (請複製並依需求修改)：

「請幫我將這 150 頁的成果報告大綱整理成『表格』格式。欄位必須包含：[投影片編號]、[標題]、[內文摘要]、[對應圖檔編號]。請以 Markdown 表格呈現。注意：標題字數請控制在 10 字以內，內文請多用明確主詞與條列式。」

※ 確保圖檔編號（如 001, 002）與您準備好的圖片檔名完全一致。

2

貼入 Google 試算表 (Sheets) 作為中轉

將 Gemini 產出的表格內容直接複製，然後開啟一份全新的 Google 試算表 並貼上。

  • 第一列 (Row 1) 應為標題列（例如：投影片編號 | 標題 | 內文摘要 | 圖片編號）。
  • 確認資料整齊排列，沒有跑位。這份試算表就是我們接下來「噴」出簡報的生產清單。
3

從試算表「一鍵生成」簡報草稿

不要傻傻地開新簡報一張張貼。我們要在試算表中執行腳本，讓它自動幫我們生成 150 頁簡報。

  1. 在您剛才的 Google 試算表中，點擊上方選單 「延伸功能」 > 「Apps Script」。
  2. 刪除編輯器中原有的程式碼，貼上以下腳本。
  3. 按下上方的 「執行 (Run)」 按鈕，並完成 Google 帳號授權。
function createSlidesFromSheet() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = sheet.getDataRange().getValues(); 
  
  // 建立一份新的簡報，名稱可自訂
  var deck = SlidesApp.create('Puti-AI 自動生成簡報草稿'); 
  
  // 從第二列開始讀取 (避開標題列)
  for (var i = 1; i < data.length; i++) { 
    var slide = deck.appendSlide(SlidesApp.PredefinedLayout.TITLE_AND_BODY);
    
    // ⚠️ 請注意陣列索引是從 0 開始算：A欄=0, B欄=1, C欄=2, D欄=3
    // 假設您的試算表中：B欄是標題，C欄是內文
    var slideTitle = data[i][1]; 
    var slideBody = data[i][2];  
    
    // 寫入標題與內文
    if (slideTitle) slide.getPlaceholder(SlidesApp.PlaceholderType.TITLE).asShape().getText().setText(slideTitle); 
    if (slideBody) slide.getPlaceholder(SlidesApp.PlaceholderType.BODY).asShape().getText().setText(slideBody);  
    
    // ⚠️ 寫入圖片標籤到備忘稿
    // 假設您的試算表中：D欄是對應圖檔編號
    var imageId = data[i][3]; 
    if (imageId) {
       var notesShape = slide.getNotesPage().getSpeakerNotesShape();
       notesShape.getText().setText("{{IMG_" + imageId + "}}");
    }
  }
  
  // 在 Apps Script 執行日誌中顯示簡報連結
  Logger.log('🎉 簡報已生成！請前往您的雲端硬碟查看，或點擊連結：' + deck.getUrl());
}

執行完畢後，去您的 Google 雲端硬碟，會看到一份全新生成、包含所有文字大綱的簡報！

💡 除錯小幫手： 如果發現生成的簡報裡面沒有標籤，或者是標題跑到內文區，請回到程式碼中檢查 `data[i][數字]`。將數字改成您試算表實際對應的欄位索引（A欄為0，B欄為1，以此類推）。
4

圖片預處理與上傳

在將圖片塞進簡報前，必須先進行「瘦身」。

  • 強烈建議： 使用 Puti-AI 圖片瘦身機，將圖片寬度統一縮放至 800px (符合 Google 簡報 16:9 比例且不卡頓)。
  • 確認圖片檔名已經按順序編號 (如 001.jpg, 002.png...)，這必須與步驟 1 大綱裡的編號一致。
  • 將處理好的圖片上傳至 Google 雲端硬碟的一個專屬資料夾，並複製該資料夾的 ID（網址列 `folders/` 後面的字串）。
5

在簡報中執行 GAS 批次置圖

打開步驟 3 生成的簡報，我們要讓程式去讀取備忘稿裡的標籤，自動把雲端資料夾裡的圖片抓進來。

常見防呆警告 (報錯：Cannot read properties of null)

請確認您已經「關閉試算表的 Apps Script」，並前往 Google 雲端硬碟 打開剛剛生成的 Google 簡報。必須在「簡報的畫面」中點選「延伸功能 > Apps Script」執行此步驟，程式才能抓到投影片喔！

  1. 在剛生成的 Google 簡報 中，點選 「延伸功能」 > 「Apps Script」。
  2. 貼入下方腳本，並記得替換資料夾 ID。
  3. 按下 「執行」，看著一百多張圖片瞬間就定位！
function batchInsertImagesByNotesTag() {
  // 🔴 請替換為您的雲端硬碟圖片資料夾 ID
  var folderId = '貼上資料夾ID'; 
  
  var deck = SlidesApp.getActivePresentation();
  var slides = deck.getSlides();
  var folder = DriveApp.getFolderById(folderId);
  var files = folder.getFiles();
  
  // 建立圖片索引地圖 (以檔名為 Key)
  var imageMap = {};
  while (files.hasNext()) {
    var file = files.next();
    var fileName = file.getName().split('.')[0];
    imageMap[fileName] = file;
  }

  // 獲取簡報寬高，用於後續排版
  var pageWidth = deck.getPageWidth();
  var pageHeight = deck.getPageHeight();

  // 逐頁檢查備忘稿是否有圖片標籤
  for (var i = 0; i < slides.length; i++) {
    var slide = slides[i];
    var notesText = slide.getNotesPage().getSpeakerNotesShape().getText().asString();
    
    // 使用簡單正則表達式尋找 {{IMG_xxx}}
    var match = notesText.match(/\{\{IMG_(.+?)\}\}/);
    
    if (match && match[1]) {
      var imgKey = match[1].trim();
      
      // 🌟 智慧防呆：解決試算表「1」與圖檔「001」不匹配的問題
      // 自動將數字補滿 3 位數 (1 -> 001)
      var paddedKey = imgKey.padStart(3, '0');
      var targetFile = imageMap[paddedKey] || imageMap[imgKey];
      
      if (targetFile) {
        var imageBlob = targetFile.getBlob();
        
        // 插入圖片
        var image = slide.insertImage(imageBlob);
        
        // --- 視覺聚焦排版設定 (Z字型) ---
        // 假設文字預設在左上方，我們將圖片置於右下方
        var newWidth = 350; // 圖片縮放寬度
        var ratio = newWidth / image.getWidth();
        var newHeight = image.getHeight() * ratio;
        
        image.setWidth(newWidth);
        image.setHeight(newHeight);
        
        // 定位：靠右 (留 20px 邊距)、靠下 (留 20px 邊距)
        image.setLeft(pageWidth - newWidth - 20);
        image.setTop(pageHeight - newHeight - 20);
      }
    }
  }
}

💡 Puti 老師的排版叮嚀與下一步

  • 套用母版： 腳本跑完後，點選簡報上方選單的「投影片」>「變更主題」，一秒鐘就能把 150 頁的背景換成您專屬的風格（建議：淡灰底 + 強調色）。
  • 視覺聚焦微調： 快速捲動檢查一遍。如果圖中人物視線朝左，記得手動將圖片移到右邊（文字放左邊），引導視覺看向重點。一張投影片保持一個重點即可！
