🎯 使用說明

  1. 首先需要取得 Gemini API Key：
    • 前往 Google AI Studio
    • 登入 Google 帳號
    • 點擊「Create API key」取得金鑰
  2. 將 API Key 輸入並儲存（僅會儲存在您的瀏覽器中）
  3. 在文字框中輸入您想要優化的貼文內容
  4. 點擊「產生貼文」，系統會自動：
    • ✨ 加入適當的表情符號
    • 🏷️ 產生相關主題標籤
    • 🌏 翻譯成英文版本
  5. 點擊「複製結果」即可將完整內容複製到剪貼板

💡 小提醒

  • 建議每次處理一則貼文內容
  • 可以透過重新整理頁面來清除預覽區的內容
  • 若遇到問題，請檢查 API Key 是否正確
